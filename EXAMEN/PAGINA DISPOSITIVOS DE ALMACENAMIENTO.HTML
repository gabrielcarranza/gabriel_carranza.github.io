DISPOSITIVOS DE ALMACENAMIENTO

Son todos los dispositivos o componentes diseñados para resguardar y almacenar información de manera tal que el sistema informático pueda tener acceso a dichos datos cuando sea necesario. En este tipo de hardwares se encuentra la memoria RAM, unidades de disco rígido, memorias flash, SSD, y otros dispositivos que cumplen el mismo fin.
